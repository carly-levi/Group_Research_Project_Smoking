Mean Girls

11/18/2015

The cases of our dataset are various countries, with a few years per country. The variables include data on enforcement of law regarding smoking, namely smoking in public locations, funding for preventing smoking, and whether or not punishments were enforced in that country during the year.

Our main interest in this data is to examine the effect of state efforts to prevent smoking. Does increased spending on prevention efforts translate to a decrease in smoking? Does this phenomena vary by gender? Does it vary by region of the world? Why?

To answer this question, several things would be useful.

  1. Visualizations

It is important to be able to visualize the relationships we are exploring. For example, a visualization of government spending as the explanatory variable and percentage of smokers as the response variable will be useful in visualizing a relationship. We will determine how appropriate a linear model can be for fitting this relationship.

  1. Hypothesis tests

It is equally important to test if differences in mean smoking rates between genders are statistically significant. We can do this through a differences in mean tests, ie. a two sample t test. We will most likely use a 95% confidence interval because that is the standard for significance.

1. Do allocating funds to enforce smoking laws make a real difference in

smoking rates? Does it vary by gender?

The first step in answering this question is to make a visualization of the variables in question, namely fundsforenforcement, Numeric, and Sex.

ggplot(data = smoking, aes(x = factor(fundsforenforcement), y = Numeric))+
  geom_boxplot()+
  facet_wrap("Sex")+
  ggtitle("Funds for Enforcement and Smoking Rates Between sexes")

We can make some initial observations from this graph. First, for both sexes, it appears that if the government allocates funds to enforce smoking laws, there are actually a greater number of people who smoke. One possible explanation for this is that in countries where funds to prevent smoking are necessary, there is already a high number of people who smoke.

Another observation we get from this plot is that smoking is much more prevalent among males than females.

The first hypothesis test we can do to test for significance is if there is a true difference in smoking rates for countries who allocate funds to prevent it or not.

We first create a bootstrap distribution of size equal to the size of the smaller sample because the originials are of different lengths.

funds_yes <- smoking %>%
  filter(fundsforenforcement == "Yes")
funds_no <- smoking %>%
  filter(fundsforenforcement == "No")
funds_boot <- data.frame(yes = rep(NA, 15000), no = rep(NA, 15000))
for(i in 1:15000){
  s <- sample(funds_yes$Numeric, length(funds_yes$Numeric), replace = TRUE)
  funds_boot$yes[i] <- mean(s)
  s <- sample(funds_no$Numeric, length(funds_no$Numeric), replace = TRUE)
  funds_boot$no[i] <- mean(s)
}

The following t test is a two sided t test to test if the sample data, not the bootstrapped data, are significantly different.

t.test(funds_yes$Numeric, funds_no$Numeric, null = 0, alternative = "two.sided")
## 
##  Welch Two Sample t-test
## 
## data:  funds_yes$Numeric and funds_no$Numeric
## t = 0.67068, df = 269.32, p-value = 0.503
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  -1.871860  3.806066
## sample estimates:
## mean of x mean of y 
##  23.56463  22.59753

Here is the t test with the bootstrapped data

se <- sd(funds_boot$yes - funds_boot$no)
p_hat <- mean(funds_yes$Numeric) - mean(funds_no$Numeric)
(boot_int = round(p_hat + c(-1,1) * 1.96 * se, 4))
## [1] -1.8238  3.7580

Here is a t test with the bootstrapped

t.test(funds_boot$yes, funds_boot$no, alternative = "two.sided")

We get a p value of 0.503, which represents the probability of getting the observed difference or more extreme, so in this case a difference of the means down to 0. The p value is relatively large, so we fail to reject the null hypothesis. Based solely on the sample data, there is not a significant difference in smoking rates between countries that provide funds for enforcement and those that do not.

From this, we gather that it might be smarter policy for countries not to spend money on enforcing smoking policies. Perhaps there is another, more efficient way to prevent people from smoking.

The next question to answer from these variables is whether or not there is a statistically significant difference between smoking rates for males and females.

The first step in answering this question is to perform a paired t test on the means for each sample.

smoking_male <- smoking %>%
  filter(Sex == "Male")
smoking_female <- smoking %>%
  filter(Sex == "Female")
sex_boot <- data.frame(male = rep(NA, 15000), female = rep(NA, 15000))
for(i in 1:15000){
  s <- sample(smoking_male$Numeric, length(smoking_male$Numeric), replace = TRUE)
  sex_boot$male[i] <- mean(s)
  s <- sample(smoking_female$Numeric, length(smoking_female$Numeric), replace = TRUE)
  sex_boot$female[i] <- mean(s)
}

Now the t test with the sample data

t.test(smoking_male$Numeric, smoking_female$Numeric, alternative = "two.sided")
## 
##  Welch Two Sample t-test
## 
## data:  smoking_male$Numeric and smoking_female$Numeric
## t = 20.041, df = 619.05, p-value < 2.2e-16
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  17.37690 21.15235
## sample estimates:
## mean of x mean of y 
##  32.53313  13.26850

Calculating the confidence interval using bootstrapped data

p_hat <- mean(smoking_male$Numeric) - mean(smoking_female$Numeric)
se <- sd(sex_boot$male - sex_boot$female)
(boot_int <- round(p_hat + c(-1,1) * 1.96 * se, 4))
## [1] 17.3639 21.1653
t.test(sex_boot$male, sex_boot$female, alternative = "two.sided")

We get a similar p value for each test, both are insignificantly different from 0.

We saw earlier from the graph that there was a fairly large disparity between male and female smoking rates, so this t test makes sense. The p value is incredibly small, so we reject the null hypothesis in favor of the alternative, the true difference in the two means is not equal to zero. We can say with 95% confidence that the true difference lies between 18.872 and 19.840.

Therefore, whatever policy the government makes to prevent smoking should be more targeted towards males because they are where the most smoking comes from.

We can now do a hypothesis test for the difference between teen and adult smoking rates.

smoking_diff <- data.frame(am_diff = rep(NA, length(smoking_americas_teens$Numeric)), af_diff = rep(NA, length(smoking_americas_teens$Numeric)))
for (i in 1:length(smoking_americas_teens$Numeric)){
  s = sample(smoking_americas_teens$Numeric, length(smoking_americas_teens$Numeric), replace = TRUE)
  s1 = sample(smoking_americas_adults$Numeric, length(smoking_americas_teens$Numeric), replace = TRUE)
  m = mean(s - s1)
  smoking_diff$am_diff[i] <- m
  s2 = sample(smoking_africa_teens$Numeric, length(smoking_africa_teens$Numeric), replace = TRUE)
  s3 = sample(smoking_africa_adults$Numeric, length(smoking_africa_teens$Numeric), replace = TRUE)
  m1 = mean(s2 - s3)
  smoking_diff$af_diff[i] <- m1
  
}

To test if the difference in means of the differences of teen and adult smoking rates are different in Africa as compared to America, we can do a paired t test on means of the differences from samples of the data.

t.test(smoking_diff$am_diff, smoking_diff$af_diff, paired = TRUE)
## 
##  Paired t-test
## 
## data:  smoking_diff$am_diff and smoking_diff$af_diff
## t = -2.9892, df = 29, p-value = 0.005649
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  -3.862269 -0.724203
## sample estimates:
## mean of the differences 
##               -2.293236

The p value of 0.00864 is low enough to reject the null hypothesis in favor of the alternative. There is a significant difference between teen and adult smoking rates in America and in Africa. This means that the difference in Africa is more accentuated, so Africa has a larger issue of teen smoking than does America.
